3.DispatcherServletAutoConfigurationиҮӘеҠЁй…ҚзҪ®еҲҶжһҗ

DispatcherServletAutoConfigurationдёӯеҗ«жңү

3.1DispatcherServletConfigurationзұ»

3.1.1 DispatcherServletConfiguration зұ»жіЁи§Јзҡ„иҜҰз»Ҷи§ЈйҮҠпјҡ

  1. @Configuration(proxyBeanMethods = false):

    • иҝҷдёӘжіЁи§ЈиЎЁзӨә DispatcherServletConfiguration жҳҜдёҖдёӘ Spring й…ҚзҪ®зұ»гҖӮ
    • proxyBeanMethods = false иЎЁзӨәиҜҘй…ҚзҪ®зұ»дёӯзҡ„ @Bean ж–№жі•дёҚдјҡиў«д»ЈзҗҶгҖӮж„Ҹе‘ізқҖ Spring дёҚдјҡдҪҝз”Ё CGLIB д»ЈзҗҶжқҘзј“еӯҳиҝҷдәӣж–№жі•зҡ„з»“жһңпјҲйҖҡеёёжҳҜдёәдәҶдјҳеҢ–жҖ§иғҪпјү

  2. @Conditional(DefaultDispatcherServletCondition.class):

    • жіЁи§ЈжҢҮзӨә DispatcherServletConfiguration зұ»зҡ„жқЎд»¶жҖ§еҠ иҪҪгҖӮеҸӘжңүеҪ“ DefaultDispatcherServletCondition жқЎд»¶ж»Ўи¶іж—¶пјҢиҝҷдёӘй…ҚзҪ®зұ»жүҚдјҡз”ҹж•ҲгҖӮDefaultDispatcherServletCondition жҳҜдёҖдёӘиҮӘе®ҡд№үжқЎд»¶зұ»пјҢз”ЁдәҺеҶіе®ҡжҳҜеҗҰеҲӣе»әиҜҘй…ҚзҪ®зұ»зҡ„ beanгҖӮиҝҷдёӘжқЎд»¶зұ»йҖҡеёёе®һзҺ°дәҶ Condition жҺҘеҸЈпјҢе№¶е®ҡд№үдәҶйҖ»иҫ‘жқҘеҲӨж–ӯжҳҜеҗҰж»Ўи¶із”ҹж•ҲжқЎд»¶гҖӮ

  3. @ConditionalOnClass(ServletRegistration.class):

    • иЎЁзӨә DispatcherServletConfiguration зұ»д»…еңЁзұ»и·Ҝеҫ„дёӯеӯҳеңЁ ServletRegistration зұ»зҡ„жғ…еҶөдёӢжүҚдјҡз”ҹж•ҲгҖӮServletRegistration жҳҜ Servlet API дёӯзҡ„дёҖдёӘзұ»пјҢеӣ жӯӨиҝҷдёӘжқЎд»¶зЎ®дҝқдәҶй…ҚзҪ®зұ»д»…еңЁеӯҳеңЁ Servlet API зҡ„зҺҜеўғдёӯиў«еҠ иҪҪгҖӮе®ғйҖҡеёёз”ЁдәҺзЎ®дҝқеҸӘжңүеңЁеҢ…еҗ« Servlet зӣёе…ізұ»зҡ„жғ…еҶөдёӢжүҚдјҡеә”з”ЁжӯӨй…ҚзҪ®гҖӮ

  4. @EnableConfigurationProperties(WebMvcProperties.class):

    • иҝҷдёӘжіЁи§ЈеҗҜз”ЁдәҶ WebMvcProperties зұ»дҪңдёәй…ҚзҪ®еұһжҖ§зҡ„ж”ҜжҢҒгҖӮиҝҷж„Ҹе‘ізқҖ Spring Boot дјҡе°Ҷ WebMvcProperties зұ»дёӯзҡ„еұһжҖ§д»Һй…ҚзҪ®ж–Үд»¶пјҲеҰӮ application.properties жҲ– application.ymlпјүдёӯз»‘е®ҡеҲ° WebMvcProperties е®һдҫӢдёӯгҖӮе®ғе…Ғи®ёиҜҘй…ҚзҪ®зұ»и®ҝй—®е’ҢдҪҝз”ЁиҝҷдәӣеұһжҖ§

    • еүҚзјҖдёәspring.mvc

з»јеҗҲжқЎд»¶

DispatcherServletConfiguration зұ»зҡ„з”ҹж•ҲжқЎд»¶жҳҜпјҡ

  1. **DefaultDispatcherServletCondition**пјҡиҮӘе®ҡд№үжқЎд»¶зұ» DefaultDispatcherServletCondition зҡ„жқЎд»¶еҝ…йЎ»ж»Ўи¶ігҖӮиҝҷдёӘжқЎд»¶зұ»йҖҡеёёз”ЁдәҺе®һзҺ°зү№е®ҡзҡ„йҖ»иҫ‘жқҘеҶіе®ҡжҳҜеҗҰеҠ иҪҪиҜҘй…ҚзҪ®зұ»гҖӮ
  2. ServletRegistration зұ»еӯҳеңЁпјҡзұ»и·Ҝеҫ„дёӯеҝ…йЎ»еҢ…еҗ« ServletRegistration зұ»гҖӮеҚіеә”з”Ёеҝ…йЎ»дҪҝз”Ё Servlet и§„иҢғпјҢе№¶дё” Servlet API зҡ„зӣёе…ідҫқиө–еҝ…йЎ»еӯҳеңЁгҖӮ
  3. WebMvcProperties й…ҚзҪ®ж”ҜжҢҒпјҡиҷҪз„¶ @EnableConfigurationProperties(WebMvcProperties.class) дёҚжҳҜз”ҹж•ҲжқЎд»¶зҡ„зӣҙжҺҘйғЁеҲҶпјҢдҪҶе®ғжҢҮзӨәеңЁй…ҚзҪ®зұ»еҠ иҪҪж—¶пјҢдјҡе°Ҷй…ҚзҪ®еұһжҖ§з»‘е®ҡеҲ° WebMvcProperties е®һдҫӢдёӯгҖӮ

з»јеҗҲжқҘиҜҙпјҢDispatcherServletConfiguration д»…еңЁж»Ўи¶і DefaultDispatcherServletCondition жқЎд»¶дё”зұ»и·Ҝеҫ„дёӯеҢ…еҗ« ServletRegistration зұ»ж—¶жүҚдјҡз”ҹж•ҲгҖӮиҝҷйҖҡеёёж„Ҹе‘ізқҖеә”з”ЁжҳҜдёҖдёӘеҹәдәҺ Servlet зҡ„ Web еә”з”ЁпјҢе№¶дё” DefaultDispatcherServletCondition жқЎд»¶жҳҜдёәдәҶиҝӣдёҖжӯҘзІҫзЎ®жҺ§еҲ¶й…ҚзҪ®зұ»зҡ„еҠ иҪҪж—¶жңәгҖӮ

3.2DispatcherServletRegistrationConfigurationзұ»

з”ҹж•ҲжқЎд»¶еҲҶжһҗ

  1. @Configuration(proxyBeanMethods = false)
    • иҝҷжҳҜдёҖдёӘж Үи®°жіЁи§ЈпјҢиЎЁзӨәиҝҷжҳҜдёҖдёӘй…ҚзҪ®зұ»пјҢз”ЁдәҺе®ҡд№ү Spring BeanгҖӮ
    • proxyBeanMethods = false иЎЁзӨәдёҚз”ҹжҲҗ CGLIB д»ЈзҗҶпјҢиҝҷжҳҜй»ҳи®Өи®ҫзҪ®пјҢдё»иҰҒз”ЁдәҺжҸҗй«ҳжҖ§иғҪгҖӮ
  2. @Conditional(DispatcherServletRegistrationCondition.class)
    • иҝҷдёӘжіЁи§ЈиЎЁзӨәиҜҘй…ҚзҪ®зұ»зҡ„з”ҹж•Ҳдҫқиө–дәҺ DispatcherServletRegistrationCondition зҡ„жқЎд»¶гҖӮ
    • DispatcherServletRegistrationCondition жҳҜдёҖдёӘиҮӘе®ҡд№үзҡ„жқЎд»¶зұ»пјҢеҸӘжңүеҪ“е…¶ matches ж–№жі•иҝ”еӣһ true ж—¶пјҢиҝҷдёӘй…ҚзҪ®зұ»жүҚдјҡз”ҹж•ҲгҖӮиҝҷдёӘжқЎд»¶зұ»зҡ„е…·дҪ“е®һзҺ°еҸ–еҶідәҺдёҡеҠЎйңҖжұӮпјҢйҖҡеёёз”ЁдәҺж №жҚ®жҹҗдәӣжқЎд»¶пјҲеҰӮзҺҜеўғгҖҒй…ҚзҪ®зӯүпјүеҶіе®ҡжҳҜеҗҰеә”з”ЁиҝҷдёӘй…ҚзҪ®гҖӮ
  3. @ConditionalOnClass(ServletRegistration.class)
    • иҝҷдёӘжіЁи§ЈиЎЁзӨәеҪ“зұ»и·Ҝеҫ„дёӯеӯҳеңЁ ServletRegistration зұ»ж—¶пјҢиҝҷдёӘй…ҚзҪ®зұ»жүҚдјҡз”ҹж•ҲгҖӮ
    • ServletRegistration жҳҜ Servlet зҡ„жіЁеҶҢзӣёе…ізұ»пјҢйҖҡеёёжҳҜдёҺ Servlet API зӣёе…ізҡ„зұ»пјҢиҝҷж„Ҹе‘ізқҖеҪ“йЎ№зӣ®дёӯжңү Servlet API зӣёе…ізҡ„дҫқиө–ж—¶пјҢиҝҷдёӘй…ҚзҪ®зұ»жүҚдјҡиў«еә”з”ЁгҖӮ
  4. @EnableConfigurationProperties(WebMvcProperties.class)
    • иҝҷдёӘжіЁи§Јз”ЁдәҺеҗҜз”ЁеҜ№ WebMvcProperties зҡ„й…ҚзҪ®еұһжҖ§ж”ҜжҢҒгҖӮ
    • WebMvcProperties жҳҜдёҖдёӘй…ҚзҪ®еұһжҖ§зұ»пјҢз”ЁдәҺд»Һй…ҚзҪ®ж–Үд»¶дёӯиҜ»еҸ– Web MVC зӣёе…ізҡ„й…ҚзҪ®пјҲеҰӮ Servlet и·Ҝеҫ„гҖҒеҗҜеҠЁеҠ иҪҪзӯүпјүгҖӮ
  5. @Import(DispatcherServletConfiguration.class)
    • иҝҷдёӘжіЁи§Јз”ЁдәҺеҜје…Ҙе…¶д»–й…ҚзҪ®зұ»гҖӮ
    • DispatcherServletConfiguration еҸҜиғҪеҢ…еҗ«дёҺ DispatcherServlet зӣёе…ізҡ„е…¶д»–й…ҚзҪ®пјҢе®ғе°Ҷдјҡиў«еҜје…ҘеҲ°еҪ“еүҚй…ҚзҪ®зұ»дёӯгҖӮ
  6. @Bean(name = DEFAULT_DISPATCHER_SERVLET_REGISTRATION_BEAN_NAME)
    • иҝҷжҳҜдёҖдёӘ Spring Bean е®ҡд№үж–№жі•пјҢиЎЁзӨәиҝ”еӣһдёҖдёӘ DispatcherServletRegistrationBean е®һдҫӢгҖӮ
    • @ConditionalOnBean жіЁи§Јз”ЁдәҺеҲӨж–ӯеҸӘжңүеңЁ DispatcherServlet Bean еӯҳеңЁж—¶жүҚеҲӣе»әиҝҷдёӘ BeanгҖӮ
  7. @ConditionalOnBean(value = DispatcherServlet.class, name = DEFAULT_DISPATCHER_SERVLET_BEAN_NAME)
    • иҝҷдёӘжіЁи§ЈиЎЁзӨәеҸӘжңүеҪ“ DispatcherServlet Bean еӯҳеңЁе№¶дё”е…¶еҗҚз§°еҢ№й…Қ DEFAULT_DISPATCHER_SERVLET_BEAN_NAME ж—¶пјҢжүҚдјҡеҲӣе»ә dispatcherServletRegistration ж–№жі•дёӯзҡ„ BeanгҖӮ
    • DispatcherServlet жҳҜ Spring MVC зҡ„ж ёеҝғ ServletпјҢеҪ“е®ғеӯҳеңЁж—¶жүҚдјҡиҝӣиЎҢжіЁеҶҢгҖӮ

иҝҷж®өй…ҚзҪ®зұ»зҡ„з”ҹж•ҲжқЎд»¶жҳҜпјҡ

  • йЎ№зӣ®дёӯеӯҳеңЁ ServletRegistration зұ»гҖӮ
  • DispatcherServletRegistrationCondition жқЎд»¶зұ»зҡ„ matches ж–№жі•иҝ”еӣһ trueгҖӮ
  • WebMvcProperties зұ»еҸҜз”ЁпјҲз”ұ @EnableConfigurationProperties жҸҗдҫӣж”ҜжҢҒпјүгҖӮ
  • DispatcherServlet Bean еӯҳеңЁпјҢе№¶дё”е…¶еҗҚз§°еҢ№й…Қ DEFAULT_DISPATCHER_SERVLET_BEAN_NAMEгҖӮ

4.HttpEncodingAutoConfiguration

4.1. й…ҚзҪ®зұ»дҪ•ж—¶з”ҹж•Ҳпјҹ

HttpEncodingAutoConfiguration й…ҚзҪ®зұ»зҡ„з”ҹж•ҲжқЎд»¶еҰӮдёӢпјҡ

  • жқЎд»¶жіЁи§Јпјҡ

    • @ConditionalOnWebApplication(type = ConditionalOnWebApplication.Type.SERVLET)

      иҝҷдёӘжіЁи§ЈзЎ®дҝқеҸӘжңүеңЁ Servlet зұ»еһӢзҡ„ web еә”з”ЁдёӯжүҚдјҡз”ҹж•ҲгҖӮ

    • @ConditionalOnClass(CharacterEncodingFilter.class)`

    иҝҷдёӘжіЁи§ЈжЈҖжҹҘ CharacterEncodingFilter зұ»жҳҜеҗҰеӯҳеңЁдәҺзұ»и·Ҝеҫ„дёӯгҖӮеҰӮжһңеӯҳеңЁпјҢй…ҚзҪ®зұ»жүҚдјҡз”ҹж•ҲгҖӮ

    • @ConditionalOnProperty(prefix = "server.servlet.encoding", value = "enabled", matchIfMissing = true)пјҡ

      иҝҷдёӘжіЁи§ЈжҢҮе®ҡдәҶдёҖдёӘеұһжҖ§еүҚзјҖе’ҢеұһжҖ§еҖјпјҲserver.servlet.encoding.enabledпјүгҖӮеҰӮжһңиҜҘеұһжҖ§еӯҳеңЁе№¶дё”еҖјдёә trueпјҢжҲ–иҜҘеұһжҖ§жңӘи®ҫзҪ®пјҢеҲҷй…ҚзҪ®зұ»з”ҹж•ҲгҖӮ

  • дҪңз”Ёпјҡ й…ҚзҪ®зұ»дјҡеңЁеҗҜеҠЁж—¶жЈҖжҹҘиҝҷдәӣжқЎд»¶гҖӮеҰӮжһңжүҖжңүжқЎд»¶йғҪж»Ўи¶іпјҢйӮЈд№Ҳ Spring Boot дјҡиҮӘеҠЁй…ҚзҪ® CharacterEncodingFilter е’Ң LocaleCharsetMappingsCustomizerгҖӮ

4.2. ејҖеҗҜй…ҚзҪ®з»‘е®ҡдәҶд»Җд№ҲпјҹдҪҝз”Ёд»Җд№ҲеүҚзјҖжқҘж“ҚдҪңзҡ„пјҹ

  • з»‘е®ҡзҡ„еұһжҖ§пјҡ

    • Encoding еҜ№иұЎпјҡй…ҚзҪ®зұ»йҖҡиҝҮ ServerProperties зҡ„ getServlet().getEncoding() ж–№жі•иҺ·еҸ– Encoding й…ҚзҪ®гҖӮиҝҷдёӘеҜ№иұЎе°ҒиЈ…дәҶеӯ—з¬Ұзј–з Ғзҡ„й…ҚзҪ®пјҢдҫӢеҰӮеӯ—з¬ҰйӣҶе’ҢејәеҲ¶зј–з Ғи®ҫзҪ®гҖӮ

  • еүҚзјҖпјҡ

    • й…ҚзҪ®зұ»дҪҝз”Ёзҡ„еүҚзјҖжҳҜ

      гҖӮдҫӢеҰӮпјҡ

      • server.servlet.encoding.charsetпјҡе®ҡд№үеӯ—з¬ҰйӣҶгҖӮ
      • server.servlet.encoding.force-requestпјҡжҳҜеҗҰејәеҲ¶иҜ·жұӮзј–з ҒгҖӮ
      • server.servlet.encoding.force-responseпјҡжҳҜеҗҰејәеҲ¶е“Қеә”зј–з ҒгҖӮ

4.3. @Bean жіЁе…ҘдәҶе“Әдәӣз»„д»¶пјҹиғҪеҗҰиҮӘе·ұжӣҝжҚўпјҹ

  • жіЁе…Ҙзҡ„з»„д»¶пјҡ

    • CharacterEncodingFilterпјҡ


- еҸҰдёҖдёӘ `@Bean` жіЁе…Ҙзҡ„з»„д»¶гҖӮе®ғе®һзҺ°дәҶ `WebServerFactoryCustomizer` жҺҘеҸЈпјҢе…Ғи®ёиҮӘе®ҡд№ү web жңҚеҠЎеҷЁзҡ„ locale е’Ң charset жҳ е°„гҖӮ

  • жҳҜеҗҰеҸҜд»ҘжӣҝжҚўпјҡ

    • **CharacterEncodingFilter**пјҡ
      • еҸҜд»ҘйҖҡиҝҮе®ҡд№үдёҖдёӘе…·жңүзӣёеҗҢ bean еҗҚз§°жҲ–зұ»еһӢзҡ„иҮӘе®ҡд№ү CharacterEncodingFilter жқҘжӣҝжҚўгҖӮ
      • з”ұдәҺ @ConditionalOnMissingBean жіЁи§ЈпјҢеҸӘжңүеңЁжІЎжңүе…¶д»– CharacterEncodingFilter bean ж—¶жүҚдјҡеҲӣе»әй»ҳи®Өзҡ„ OrderedCharacterEncodingFilterгҖӮ
    • **LocaleCharsetMappingsCustomizer**пјҡ
      • еҸҜд»ҘйҖҡиҝҮе®ҡд№үдёҖдёӘиҮӘе®ҡд№үзҡ„ WebServerFactoryCustomizer bean жқҘжӣҝжҚўгҖӮ
      • иҮӘе®ҡд№үе®һзҺ°еҸҜд»Ҙи°ғж•ҙ locale е’Ң charset жҳ е°„йҖ»иҫ‘пјҢе…Ғи®ёејҖеҸ‘иҖ…ж №жҚ®йңҖжұӮиҝӣиЎҢй…ҚзҪ®гҖӮ
