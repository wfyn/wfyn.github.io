Collectors API

发表于2024-05-26|更新于2024-05-26|java
|阅读量:

Collectors API 是 Java 中用于收集（collecting）流元素到各种数据结构的工具类。它提供了丰富的静态方法来支持将流元素收集到列表、集合、映射等不同类型的数据结构中。这些方法可以与流的 collect 方法一起使用，用于生成最终的结果。

下面是一些常用的 Collectors API 的用法示例：

1. toList()

将流中的元素收集到一个列表中。

2. toSet()

3. toMap()

将流中的元素收集到一个映射中，可以指定键和值的映射关系。

示例：

4. joining()

将流中的元素连接成一个字符串。

示例：

5. groupingBy()

根据指定的分类函数对流中的元素进行分组。

示例：

6. partitioningBy()

根据给定的条件将流中的元素分成两组。

示例：

7. counting()

计算流中元素的个数。

示例：

8. averagingInt()

计算流中整数类型元素的平均值。

示例：

9. summarizingInt()

对流中的整数类型元素进行汇总统计，包括计数、总和、平均值、最大值和最小值。

10. joining()

将流中的元素连接成一个字符串，可以指定连接符、前缀和后缀。
